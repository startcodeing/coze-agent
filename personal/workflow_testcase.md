# 隐含风格语义（模型需理解语境）
```
像宫崎骏动画一样温暖治愈的故事
画面像老电影一样有胶片颗粒感
整个世界充满霓虹灯和机械义体
画面像童话绘本一样温馨可爱
人物造型夸张可爱，色彩鲜艳明亮
整个故事充满未来科技和人工智能元素
```
